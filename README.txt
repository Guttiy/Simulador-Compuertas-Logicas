		Simulador-Compuertas-Logicas

- Para correr el programa entrar a la carpeta Ejecutable y correr Simulador.jar, 
o desde eclipse.

	* Para correr desde eclipse en windows es necesario tener java y configurar las 
	variables de entorno, para saber como, dirigirse a la siguiente direccion 
	en el video mas o menos en el minuto 4:15:
	http://hypsoluciones.com/2015/08/java-8-ambiente-de-trabajo-con-eclipse-mars/ 
	* Para correr el ejecutable en distribuciones linux ejecutar desde la terminal 
	./Simulador.jar, o puede remitire al siguiente articulo y seguir los pasos alli 
	indicados:
	https://andalinux.wordpress.com/2008/12/17/ejecutar-aplicaciones-jar-en-ubuntu-linux/

- Una vez dentro del programa en el menu CREAR se encuentran las compuertas y cables, 
y dentro del menu ENT/SAL se encuentran los valores 0-1 para las entradas y la salida, 
salida se refiere al valor de salida de una compuerta que se quiere simular.

- Al momento de colocar las compuertas tener en cuenta que el puntero del mouse indica 
la esquina superior izquierda de donde colocara la compuerta.

- Al momento de colocar un cable, una entrada o una salida debe localizarse el puntero 
del mouse un poco a la izquierda y un poco arriba del pin de la compuerta. Ademas para 
un cable debe arrastrase el mouse de un pin a otro.

- El boton GUARDAR permite guardar el diseño que se ha creado hasta ese momento, por 
ahora solo permite guardar un diseño, para volverlo a abrir despues de cerrado el 
programa se debe hacer clic en ABRIR.

- Si se como algun error al colocar un elemento debera hacer clic en el boton LIMPIAR, 
por ahora solo es posible limpiar completamente el espacio de trabajo, no un solo elemento. 
	* Para que este boton funcione es necesario que el archivo Limpio.txt se encuentre 
	en la misma carpeta que Simulador.jar, o donde esta el proyecto de eclipse, tal y 
	como se encuentra en el repositorio.
